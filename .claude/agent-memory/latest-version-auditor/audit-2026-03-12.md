## Key Findings

- `qrcode` + `@types/qrcode`: zero imports in codebase, safe to remove
- `execa`: used only in lib/exec.ts, replaceable with node:child_process (12 transitive deps)
- `@11ty/gray-matter`: used only in lib/frontmatter.ts parseFrontmatter(), replaceable with js-yaml + ~15 lines (3 transitive deps)
- `matrix-bot-sdk`: 0.7.1 -> 0.8.0 available, requires Node >=22 (already satisfied)
- `hosted-git-info` + `@types/hosted-git-info`: used in one function (lib/git.ts), could be replaced by regex but handles many URL edge cases
- Biome used via npx, not declared as dependency (intentional)
- Pi SDK packages correctly kept as peerDependencies
